## 💡 IMPORTANT NOTES

### About Environment Variables (.env files)

✅ All `.env` files are in `.gitignore` - they won't be committed to GitHub
✅ Never commit secrets to GitHub
✅ On Render/Vercel, set environment variables in the dashboard
✅ Different values for development vs production

### About Free Tier Services

**Render Free Tier:**
- Backend goes to sleep after 15 mins of inactivity
- If you get 502 error, just click the service in Render to wake it up
- This is normal and expected

**Vercel Free Tier:**
- Frontend always available
- No cold starts like Render
- Perfect for React apps
